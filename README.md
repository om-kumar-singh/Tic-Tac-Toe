# Tic-Tac-Toe Android App

A clean, classic Tic-Tac-Toe built with Android Studio (Java), now updated with a simple Light theme and improved app icon.

## App
- Package: com.example.tictactoev4
- Min SDK: 21 (Lollipop)
- Build: Gradle (Kotlin DSL)

## Features
- Single-player (basic AI) and Two-player modes
- Minimal, friendly Classic Light UI
- Adaptive app icon

## Run Locally
1. Open the project in Android Studio (Giraffe+ recommended)
2. Let Gradle sync
3. Run the pp module on an emulator or device

## Build APK
- Android Studio: Build > Build Bundle(s)/APK(s) > Build APK(s)
- CLI: ./gradlew assembleDebug

## Folder Structure
- pp/ – Android module with Activities, layouts, and resources
- pp/src/main/java/com/example/tictactoev4/ – Java code (MainActivity, GameActivity)
- pp/src/main/res/ – Layouts, drawables, colors, themes, and icons
