## ðŸ•¹ï¸ How to Play

1. The game is played on a 3x3 grid
2. Players take turns placing X and O marks
3. First player to get 3 of their marks in a row (horizontal, vertical, or diagonal) wins
4. If all 9 squares are filled without a winner, the game is a draw
5. Tap the "Reset Game" button to start a new game
